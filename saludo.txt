Listado de comandos:
Git init = preparar un unevo repositorio
git remote add <nombre_corto> <url> = Para interactuar con un repositorio remoto en particular
Git clone <url> = Clonar
Git add <archivo> = Añadir un nuevo archivo
Git commit -m <Mensaje> = Añadir un mensaje
Git push origin <branch> = Especificar a que parte subir 
git remote set-url origin <url> = Para cambiar la URL del repositorio remoto.
git pull origin <branch> = Para actualizar su repositorio local con los cambios del repositorio remoto
git remote -v = Si desea ver una lista de todos los repositorios remotos que ha agregado a su repositorio local
git branch = Saber que rama estamos posicionados
git branch <branch> = Se crea una nueva rama
git branch -m <nombreBranchActual> <nombreBranchNueva> = Cambiar de nombre de una rama
git checkout <branch> = Cambiar de rama
git branch -d <branch> = Borrar una rama
git diff <branch> <branch> = Checar diferencias de ramas
git merge <branchOrigen> <branchDestino> = Unificar dos ramas
